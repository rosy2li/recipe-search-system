# 食光菜谱 V48 数据库驱动版说明

## 一、版本定位

V48 是在 V47 MySQL 数据库基础上的数据库驱动版。系统启动后，`server.py` 连接 MySQL，并通过 `/api/bootstrap` 向前端提供菜谱数据。前端不再依赖 `src/data/recipes.js`。

## 二、数据流

```text
浏览器页面
→ GET /api/bootstrap
→ server.py
→ MySQL recipe 数据库
→ 返回菜谱、食材、步骤、图片、标签
→ 前端渲染检索、详情、套餐、AI 问菜
```

## 三、保留能力

- 菜谱检索
- 多条件筛选
- 菜谱详情页
- 收藏
- 营养套餐
- AI 问菜
- MySQL 数据接口测试

## 四、运行要求

必须先导入 V47 的数据库初始化 SQL，并在 `.env` 中配置 MySQL 连接信息。
